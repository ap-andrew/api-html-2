Developer and App Administration

Developers

/developers
A developer who create apps that consume provider APIs
Developers can register themselves through a developer portal when they want to access your APIs, but there are cases when you would add a developer via RESTful API. For example, your API could be for use by internal developers and you might not have a developer portal, or you may wish to programmatically add a large number of developers as part of a migration. Developers build apps that access your APIs. Apps are provisioned with keys and secrets, which in turn provide access to the API Products configured for your organization.
