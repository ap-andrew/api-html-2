Delete an API Poduct :: /apiproducts/{apiproductName}

API resource: API Products

Verb: DELETE

Doc: Deletes an API product from an organization. Note that deleting an API product will cause app requests to the resources URIs) defined in the API product to fail. Make sure you create a new API product to serve existing apps (unless your intention is disable access to the resources defined in the API product.)

Description: Deletes an API Product

cURL: 
curl -u {myname:mypass} "https://api.enterprise.apigee.com/v1/o/{myorg}/apiproducts/{apiproductName}"
