Create App for a Company :: /companies/{companyname}/apps

API resource: Company Apps

Verb: POST

Doc: Creates an app that belongs to a company.

Description: Creates an App for a Company

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/organizations/{organizationname}/companies/{companyName}/apps -X POST
