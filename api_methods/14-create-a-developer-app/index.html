Create a Developer App :: /developers/{developerName}/apps

API resource: Developer Apps

Verb: POST

Doc: Optional elements: AccessType, CallbackUrl, ApiProducts, Attributes Note: CallbackUrl can be ‘,’ separated multiple callback URLs’.

Description: Creates an App associated with a Developer

cURL: 
curl -u {myname:mypass} "https://api.enterprise.apigee.com/v1/o/{myorg}/developers/{developerName}/apps/"
